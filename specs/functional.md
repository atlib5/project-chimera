## Actors
- Coordinator Agent
- Worker Agents (Trend Research, Content Generation, Engagement Tracking)
- Human Reviewer

## User Stories

### Trend Research
- As an Agent, I need to fetch trending topics from external platforms
  so that content ideas are relevant and timely.

### Content Generation
- As an Agent, I need to generate content drafts based on trends
  so that they can be reviewed before publishing.

### Human Review
- As a Human Reviewer, I need to approve or reject generated content
  so that unsafe or low-quality content is not published.

### Publishing
- As an Agent, I need to publish approved content automatically
  so that engagement can be tracked efficiently.

### Engagement Tracking
- As an Agent, I need to collect engagement metrics
  so that future content decisions can be improved.
